# Graph Algorithm Visualizer

Standalone React/Vite app that uses the published `nodes-connector` package to visualize graph algorithms.

## Run

```bash
npm install
npm run dev
```

Open:

```txt
http://127.0.0.1:5174
```

## Algorithms

- Breadth-first search
- Depth-first search
- Dijkstra shortest path

The graph board is rendered with `nodes-connector/react`.
